

## Definition

  

- Delegates are **type-safe function pointers** in C\# that reference methods with a specific signature.

- Types of Delegates:

    - **Singlecast Delegate**: Points to a single method.

    - **Multicast Delegate**: References and invokes multiple methods sequentially.

    - **Generic Delegate**: Defined with type parameters for reusability, e.g., `Func<>`, `Action<>`.

    - **Anonymous Method Delegate**: Delegate instantiated with an anonymous method.

    - **Lambda Expression Delegate**: Delegate instantiated with a lambda expression for concise inline methods.

  

***

  

## 2. Anonymous Methods, What is Events?

  

### Anonymous Methods

  

- Introduced in C\# 2.0.

- Allow defining inline unnamed methods assigned directly to delegates.

- Syntax uses the `delegate` keyword followed by code block.

  

Example:

  

```csharp

MyDelegate del = delegate(string msg)

{

    Console.WriteLine("Anonymous: " + msg);

};

del("Hello");

```

  
  

### Events

  

- Events are **special delegates** enabling a publisher-subscriber pattern.

- They allow objects to notify other objects when something happens.

- Declared using the `event` keyword.

- Events restrict delegate invocation access, only allowing the declaring class to invoke the event.

  

Example:

  

```csharp

public delegate void Notify(string message);

public event Notify OnNotify;

  

public void RaiseEvent()

{

    OnNotify?.Invoke("Event triggered");

}

```

## 3. Multicast Events, Lambda Expression

  

### Multicast Events

  

- A delegate instance that has reference to multiple methods.

- Use `+=` operator to add methods, `-=` to remove.

- Invokes all methods in invocation list in order.

  

Example:

  

```csharp

MyDelegate del = MethodA;

del += MethodB;

del("Hello"); // Calls MethodA and MethodB

```

  
  

### Lambda Expressions

  

- Short, inline anonymous functions.

- Often used with delegates and events for concise code.

  

Example:

  

```csharp

MyDelegate del = (msg) => Console.WriteLine("Lambda: " + msg);

del("Hello Lambda");

```

## 4. How to Use Delegates in C\# with Example: Button Click Event

  

### Using Delegates for Events in WinForms

  

- Windows Forms buttons expose `Click` events.

- You attach event handler methods or lambdas to these events, which are multicast delegates.

  

Example:

  

```csharp

using System;

using System.Windows.Forms;

  

class MyForm : Form

{

    Button myButton;

  

    public MyForm()

    {

        myButton = new Button

        {

            Text = "Click Me",

            Width = 100,

            Height = 50,

            Top = 20,

            Left = 20

        };

  

        // Attach event handler using delegate method

        myButton.Click += MyButton_Click;

  

        // Attach event handler using lambda expression

        myButton.Click += (sender, e) => MessageBox.Show("Clicked using lambda!");

  

        Controls.Add(myButton);

    }

  

    private void MyButton_Click(object sender, EventArgs e)

    {

        MessageBox.Show("Button Clicked!");

    }

  

    [STAThread]

    public static void Main()

    {

        Application.EnableVisualStyles();

        Application.Run(new MyForm());

    }

}

```
