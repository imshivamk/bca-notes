### OPERATING SYSTEM

1. The operating system is the most <mark style="background: #FFB8EBA6;">prominent</mark> example of system software that acts as an <mark style="background: #FFB86CA6;">interface between the user and system hardware</mark>.
2. It is the <mark style="background: #FFB8EBA6;">main program of a computer system</mark>.
3. When the computer system ON it is the <mark style="background: #FFF3A3A6;">first software that loads</mark> into the computer’s memory.
4. Basically, it <mark style="background: #FFF3A3A6;">manages all the resources such as computer memory, CPU, printer, hard disk, etc.,</mark>
5.  It <mark style="background: #FFF3A3A6;">provides an interface to the use</mark>r, which helps the user to interact with the computer system.
6. It also provides <mark style="background: #ABF7F7A6;">various services to other computer software</mark>. Examples of operating systems are Linux, Apple macOS, Microsoft Windows, etc

### LANGUAGE PROCESSORS

2. So, the <mark style="background: #FFB86CA6;">conversion is done by the language processor.</mark>
