1. The WordPress Dashboard is a <mark style="background: #FFB8EBA6;">first screen which will be seen when you log in, which will display the overview of the website.</mark>
2. It is a <mark style="background: #FFF3A3A6;">collection of gadgets</mark> that provide information and an overview of what's happening with your blog.
3. You can customize your needs by using some quick links such as <mark style="background: #FFB8EBA6;">writing quick draft,</mark> <mark style="background: #FFB8EBA6;">replying to latest comment</mark>, etc.

Dashboard can be categorized as-



1. **Dashboard Menu**
- <mark style="background: #FFF3A3A6;">navigation menu that contains some menu options such as posts, media library, pages, comments, appearance options, plugins, users, tools and settings on the left side.</mark>

2. **Screen Options**
- The dashboard contains <mark style="background: #ADCCFFA6;">different types of widgets which can be shown or hidden on some screens.</mark>
- It contains <mark style="background: #FFF3A3A6;">check boxes to show or hide screen options and also allows us to customize sections on the admin screen.</mark>

3. **Welcome**
1. It includes the<mark style="background: #D2B3FFA6;"> Customize Your Site button</mark> which allows customizing your WordPress theme.
2. The center column provides some of the useful links such as <mark style="background: #ADCCFFA6;">creating a blog post, creating a page and view the front end of your website.</mark>
3. <mark style="background: #FFF3A3A6;">Last column contains links to widgets, menus, settings related to comments</mark> and also a link to the First Steps With WordPress page in the WordPress codex.

4. **Quick Draft**
- mini post editor which allows writing, saving and publishing a post from admin dashboard.
- It includes the title for the draft, some notes about the draft and save it as a Draft.

5. **WordPress News**
- widget displays the latest news such as <mark style="background: #FFF3A3A6;">latest software version, updates, alerts, news regarding the software etc</mark>. from the official WordPress blog.

6. **Activity** (NOTIFICATIONS)
- The Activity widget includes <mark style="background: #FFF3A3A6;">latest comments on your blog, recent posts and recently published posts.</mark>
- It allows you to <mark style="background: #FFB86CA6;">approve, disapprove, reply, edit, or delete a comment.</mark>
- It also allows you to <mark style="background: #D2B3FFA6;">move a comment to spam.</mark>

7. **At a Glance**
- gives an <mark style="background: #ADCCFFA6;">overview of your blog's posts, number of published posts and pages, and number of comments.</mark>
- When you click on these links, you will be taken to the respective screen.
- It displays the <mark style="background: #FFF3A3A6;">current version of running WordPress</mark> along with the currently running <mark style="background: #FFF3A3A6;">theme on the site.</mark>
