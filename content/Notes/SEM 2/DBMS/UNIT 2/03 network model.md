### Features of a Network Model  
1. <mark style="background: #FF5582A6;">Multiple Paths</mark>- There can be more than one path to the same record because of more relationships. It makes data access fast and simple.
2. <mark style="background: #ADCCFFA6;">Ability to Merge Relationships</mark> − In this model, because of more relationships the data is more related. It has an ability to manage one-to-one relationships as well as many-to-many relationships.
3. <mark style="background: #BBFABBA6;">Circular Linked List</mark> − The operations in this model are done with the help of the circular linked list. The <mark style="background: #CACFD9A6;">current position is maintained</mark> with the help of a program and <mark style="background: #CACFD9A6;">navigates through the records based on relationships.</mark>
