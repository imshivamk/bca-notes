## Data processing

1. Process of <mark style="background: #D2B3FFA6;">performing specific operations on a set of data</mark> or a database. 
2. A database is an<mark style="background: #D2B3FFA6;"> organized collection of facts and information, such as records on employees, inventory,</mark> customers, and potential customers.
3. <mark style="background: #D2B3FFA6;">collecting raw data and translating it into usable information.</mark>
4. The raw data is <mark style="background: #D2B3FFA6;">collected, filtered, sorted, processed, analyzed, stored</mark>, and then <mark style="background: #ABF7F7A6;">presented in a readable format</mark>.
5. The data processing is carried out <mark style="background: #FFF3A3A6;">automatically or manually.</mark>
6. automatic : using computers - faster and accurate results
7. data can be converted into <mark style="background: #FFF3A3A6;">different forms</mark>. graphic, audio etc
8. It depends on the software used as well as data processing methods.
9. Data processing is crucial for organizations to create better <mark style="background: #FFB8EBA6;">business strategies</mark> and increase their<mark style="background: #FFB8EBA6;"> competitive edge.</mark>
10. <mark style="background: #D2B3FFA6;">By converting the data into a readable format like graphs, charts, and documents, employees throughout the organization can understand and use the data</mark>

## stages of data processing

### 1. Data Collection
1. <mark style="background: #FFB8EBA6;">first step</mark> of the processing cycle.
2. The <mark style="background: #ADCCFFA6;">raw data</mark> collected has a <mark style="background: #FFF3A3A6;">huge impact on the output</mark> produced.
3. Raw data should be gathered from <mark style="background: #FFB86CA6;">defined and accurate sources</mark> so that the <mark style="background: #ABF7F7A6;">subsequent findings are valid and usable.</mark>
4. Raw data : monetary figures, website cookies, profit/loss statements of a company, user behavior, etc.
### 2. Data Preparation
1. <mark style="background: #2dba4e;">sorting and filtering </mark>the raw data to <mark style="background: #2dba4e;">remove unnecessary and inaccurate data.</mark>
2. Raw data is <mark style="background: #2dba4e;">checked for errors, duplication, miscalculations, or missing data </mark>and transformed into a<mark style="background: #ABF7F7A6;"> suitable form for further analysis and processing. </mark>
3. This ensures that only the highest quality data is fed into the processing unit.
### 3. Data Input
1. raw data is <mark style="background: #FFB86CA6;">converted into machine-readable form and fed into the processing unit.</mark>
2. Form of <mark style="background: #FFB8EBA6;">data entry through</mark> a keyboard, scanner, or any other <mark style="background: #FFB8EBA6;">input source</mark>.
### 4. Data Processing
1. data is subjected to various data <mark style="background: #D2B3FFA6;">processing methods</mark> using <mark style="background: #FFB86CA6;">machine learning and artificial intelligence algorithms</mark> to <mark style="background: #D2B3FFA6;">generate the desired output.</mark>
2. This<mark style="background: #ADCCFFA6;"> step may vary slightly from process to process depending on the source of data </mark>being processed (data lakes, online databases, connected devices, etc.) and the intended use of the output. 
### 5. Data Interpretation or Output 
1. The data is finally transmitted and displayed to the user in a readable form like graphs, tables, vector files, audio, video, documents, etc. - This output can be stored and further processed in the next data processing cycle.
### 6. Data Storage
1. The last step of the data processing cycle is storage,
2. Where data and metadata are stored for further use.
3. This allows quick access and retrieval of information whenever needed
